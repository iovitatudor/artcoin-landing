## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

```
npm run build
```

### Lints and fixes files

```
npm run lint
```

## Run With Docker Compose:

### build with no cache

```
docker-compose build --no-cache
```

### start the services

```
docker-compose up
```

### stop services

```
docker-compose stop
```
